------------
<b>How to install</b>
<p>Clone this repository:<br/>
git clone git@github.com:redVi/dwm.git<br/>
<p>Go to the directory and build the package:</p>
cd dwm/<br/>
makepkg PKGBUILD<br/>
pacman -U dwm-6.0-1-**.pkg.tar.xz<br/>
restart<br/>
